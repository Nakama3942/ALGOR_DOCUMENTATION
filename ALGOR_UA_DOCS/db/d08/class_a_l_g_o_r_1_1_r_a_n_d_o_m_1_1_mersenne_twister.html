Детальний опис

Складний ГПВЧ, що базується на властивостях простих чисел Мерсенна

Вихор Мерсенна — ГПВЧ, розроблений у 1997 році японськими вченими Макото Мацумото та Такудзі Нісімурою. Вихор Мерсенна ґрунтується на властивостях простих чисел Мерсенна та забезпечує швидке генерування високоякісних за критерієм випадковості псевдовипадкових чисел.

Починаючи з
v0.1.0.0 commit 88415c
Не рекомендовано:
На даному етапі розробка даного ГВЧ заморожена. Нехай він і краще за Лінійні ГПВЧ, його ще не доведено до відлагодженого стану. Його ще й досі складно використовувати. У майбутньому можлива повна зміна сигнатур методів у класі. Чи його повне видалення.

Метод генеації даних на усьому можливому діапазоні

Генерує числа на увесь можливий діапазон змінної. Цей метод спеціально було створено під сумісність з функцією тестування ГПВЧ.

Починаючи з
v0.1.3.0 commit ca0704
Застереження
Цей метод було написано лише для функції тестування ГПВЧ. Не використовуйте його, якщо у вас є обмеження на число, що має бути згенероване. Для цього використовуйте метод ALGOR::RANDOM::MersenneTwister::IRandom, що спеціально для цього й створювався.
А також, якщо Вам потрібно згенерувати число на усьому діапазоні, то краще використати ALGOR::RANDOM::MersenneTwister::BRandom, щоб уникнути додаткових розрахунків.
Не рекомендовано:
Буде видалено після завершення тестування ГПВЧ.
Повертає
згенероване число

ALGOR::RANDOM::MersenneTwister::BRandom ( )

Метод генерації випадкових бітів

Генерує випадкові біти у змінній з чотирьох байт, що дозволяє згенерувати 32-бітове число на всьому діапазоні.

Починаючи з
v0.1.0.0 commit 88415c
Нотатки
Може слугувати аналогом методу ALGOR::RANDOM::MersenneTwister::rand, але той метод є надстройкою над ALGOR::RANDOM::MersenneTwister::IRandom для функції тестування ГПВЧ, що викликає додаткові розрахунки та витрати ресурсів. Якщо потрібно відразу згенерувати якесь число на усьому діапазоні, то краще обирати цей метод.

згенероване число на всьому діапазоні
