Детальний опис

Клас-каркас для роботи з винятками

Клас, що зберігає данні про виняток (номер винятка, деталі та пояснення). Щоб створити об'єкт винятку достатньо переадти необхідні данні конструктору, а для отримання інформації про виняток треба його перехопити і використати метод, що повертає необхідні данні.

Починаючи з
v0.1.1.0 commit c62e85
Тест:
try
{
throw ALGOR::EXCEPTION_SET::Exception(55, "details", "explanation");
}
catch (ALGOR::EXCEPTION_SET::Exception ex)
{
std::cout << "Code: " << ex.code() << ";\n" << ex.what() << "\n-> " << ex.why() << "\n";
}
// >print:>
// Code: 55;
// details
// -> explanation
